git基本操作。

配置git

1
2

$ git config --global user.name "yemo"    // 用户名和账号名一致
$ git config --global user.email "10000@qq.com"    // 邮箱

查看是否配置成功

1

 git config -l

初始化版本库

1

git init

添加暂存区 & 本地库

1
2

git add <file> // 添加到缓存区
git add . // 全部提交到暂存区

// 创建分支
git branch [版本名称]

// 查看分支
git branch -v

// 查看全部分支
git branch -a

// 切换分支
git checkout [分支名称]

// 吧指定的分支合并到当前分支
git merge [分支名称]

// 删除本地分支
git branch -d [分支名称]

// 强制删除本地分支
git branch -D [分支名称]

// 删除远程分支
git push origin --delete [分支名称]

    -t：指定要创建的密钥类型。

    -l：显示公钥文件的指纹数据； 
    -N：提供一个新密语； 
    -P：提供（旧）密语；
    -q：静默模式；

bug

warning: LF will be replaced by CRLF in

1
2
3
4
5

原因是存在符号转义问题

windows中的换行符为 CRLF， 而在linux下的换行符为LF，所以在执行add . 时出现提示，解决办法：

git config --global core.autocrlf false
