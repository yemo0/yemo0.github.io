vue 项目@没路径提示

May 29, 2022 · 2 mins read

vue 项目使用@路径提示。

  1. 在项目下面创建jsconfig.json

  2. 写入内容

    {
    "compilerOptions": {
        "baseUrl": ".",
        "paths": {
            "@/*": [
                "./src/*"
            ]
        }
    },
    "exclude": [
        "node_modules",
        "dist"
    ]
}
