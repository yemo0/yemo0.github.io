git push到多个远程仓库

May 29, 2022 · 1 mins read

执行git push同时提交到两个仓库。

方法1

git remote set-url --add [别名] [仓库地址]

方法2

修改.git/config文件，类似下面这样：

[remote "gitlab"]
	url = git@gitlab.com:yemo0/xxx.git
	fetch = +refs/heads/*:refs/remotes/gitlab/*
    url = git@github.com:yemo0/xxx.git
