# linux常用命令


### 解压

```
tar -C /usr/local -xzf go1.4.linux-amd64.tar.gz
```

## 系统

- arch	显示处理器架构
- uname -m 显示处理器架构
- uname -r 显示正在使用的内核版本
- cat /proc/cpuinfo CPU info 信息
- cat /proc/interrupts 显示中断

## 1 文件目录操作

### ls

查看文件夹文件

##### 常用参数

- -l   列出数据串、文件属性、和权限
- -a  列出全部文件、包含隐藏文件
- -R   递归列出

### mkdir

创建目录

常用参数

- -m <权限> <文件夹名>
- -v 每次创建新目录都显示信息

### rm

删除

#### 常用参数

- -f 强制删除 忽略不存在的文件
- -i 交互式删除
- -r 递归删除

### mv

移动、改名

#### 常用参数

- -f 强制
- -i 如果目标文件存在，询问是否覆盖
- -u 若目标文件存在。且source比较新才会更新
